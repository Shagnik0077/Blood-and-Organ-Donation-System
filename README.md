# 🩸 Blood and Organ Donation System

A full-stack web application designed to streamline and manage **blood and organ donation processes**, enabling donors to register, medical staff to verify eligibility, and organizations to efficiently connect donors with recipients.

---

## 📌 Project Overview

The **Blood and Organ Donation System** is built to solve real-world challenges in donor management and awareness. It provides a centralized platform where:

- Blood donors can easily register and donate  
- Organ donors undergo proper medical screening  
- Healthcare organizations can manage donor data efficiently  

This system ensures a **safe, transparent, and reliable donation workflow**.

---

## 🚀 Features

### 🧑‍🤝‍🧑 Donor Management
- Blood donor registration and profile management  
- Organ donor registration with detailed medical data  
- Secure storage of donor information  

### 🏥 Medical Verification
- Organ donors undergo:
  - Heart health checks  
  - Lung condition evaluation  
  - Medical eligibility verification  
- Ensures safe transplantation  

### 🔍 Search & Matching
- Search donors based on:
  - Blood group  
  - Location  
  - Availability  
- Quick access during emergencies  

### 📊 Admin Panel
- Manage donor records  
- Approve and verify registrations  
- Monitor donation activities

## ⚙️ System Workflow

1. User registers as a **blood or organ donor**  
2. Data is stored in the database  
3. Organ donors go through medical verification  
4. Admin approves eligible donors  
5. Donors become available in the system  
6. Users search and find suitable donors  
7. Donation process is coordinated safely
